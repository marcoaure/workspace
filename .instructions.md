## Convenções deste repo

1. **Cross-platform:** `.ps1` para Windows, `.sh` para Mac/Linux. Nunca misturar.
2. **Zero dependências externas quando possível** — preferir APIs nativas do OS.
3. **Cada módulo é independente** — próprio README.md e .instructions.md.
4. **config em JSON** — fácil de ler e editar tanto por humanos quanto por scripts.
5. **PowerShell aliases** registrados via `setup.ps1` de cada módulo no `$PROFILE`.
6. **Mac aliases** registrados via `setup.sh` de cada módulo no `.zshrc`.

## Ao adicionar um novo módulo

1. Criar pasta com nome descritivo (kebab-case)
2. Incluir `README.md` com docs completos
3. Incluir `.instructions.md` com contexto para IA
4. Se tiver setup, criar `setup.ps1` (Win) e/ou `setup.sh` (Mac)
5. Atualizar o README.md raiz com o novo módulo
